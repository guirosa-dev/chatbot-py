# Chatbot-PY

Um chatbot simples desenvolvido em Python utilizando Streamlit e a API do Google Gemini, permitindo que o usuário converse com uma inteligência artificial diretamente em uma interface web.

### Sobre
 - O chatbot-py é um projeto desenvolvido com o objetivo de praticar integração com APIs de inteligência artificial e desenvolvimento de interfaces web simples em Python.

 - A aplicação permite que o usuário envie mensagens por meio de uma interface de chat, enquanto o sistema utiliza o modelo Gemini para gerar respostas automaticamente.

 - O projeto também utiliza gerenciamento de estado do Streamlit para manter o histórico da conversa durante a execução da aplicação.

 - Este projeto foi criado como parte de estudos na área de desenvolvimento de software e integração com IA.

## Como Executar
### 1° - Clone o repositório
```
git clone https://github.com/guirosa-dev/chatbot-py
```
### 2° - Entre na pasta do projeto
```
cd chatbot-py
```
### 3° - Instale as dependências
```
pip install streamlit
pip install google-generativeai
```
### 4° - Adicione sua chave da API do Gemini no código
```
genai.configure(api_key="SUA_API_KEY")
```
### 5° - Execute o projeto
```
streamlit run app.py
```
